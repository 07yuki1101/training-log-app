## トレーニングログアプリ
個人で筋力トレーニングをする中で、
手軽に記録でき振り返りやすいツールがなく、
自分が使いやすいトレーニングログを作りたいと思い制作しました。

## 主な機能
- Firebase Authenticationによるログイン機能
- トレーニング記録の登録・編集・削除
- 日付ごとのトレーニングログ管理

## 使用技術
- React
- Vite
- Firebase（Authentication / Firestore）
- CSS
- Vercel

## 工夫した点
- フォーム入力時に当日の日付が自動で入るようにし、入力の手間を減らした
- Firestoreのデータ構造をシンプルに設計し、管理しやすさを意識した
- 体重の推移をグラフ化して一目で増減がわかるようにした
